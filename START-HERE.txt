🐛 TROUBLESHOOTING

Problem: "Connection Refused" when opening http://localhost:5174
→ Solution: Wait 5 seconds, then refresh (Ctrl+R)
→ OR check both server windows are open

Problem: "Port already in use"
→ Solution: netstat -ano | findstr :5000
→ Then: taskkill /PID [number] /F

Problem: "MongoDB Connection Failed"
→ Solution: Connection is configured, MongoDB may be offline
→ The system will auto-retry

Problem: Dependencies won't install
→ Solution: Delete node_modules and try again
→ Command: rmdir /s node_modules && npm install

See QUICK-START.md for more detailed troubleshooting
