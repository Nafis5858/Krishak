## 🐛 If You Encounter Issues

### "Connection Refused"
- Make sure both servers are running
- Wait 5 seconds and refresh browser
- Check if ports 5000 and 5174 are available

### "MongoDB Connection Failed"
- This is normal if internet is slow
- MongoDB Atlas is cloud-based
- System will auto-retry connection

### "Port Already in Use"
```batch
netstat -ano | findstr :5000
taskkill /PID [number] /F
```

### "npm install Failed"
```batch
cd f:\Krishak-main\backend
rmdir /s /q node_modules
del package-lock.json
npm install
```
